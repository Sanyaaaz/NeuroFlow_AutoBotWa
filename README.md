# NeuroFlowOS

NeuroFlowOS is a responsive and modern landing page built using Next.js, Tailwind CSS, and Vercel v0. It showcases a futuristic cognitive performance platform with a clean UI, adaptive layouts, and a well structured component architecture. The project demonstrates a production ready design with strong attention to responsiveness, layout balance, and accessibility.

## Features

- Fully responsive landing page built with Tailwind CSS  
- Clean component structure following Next.js best practices  
- Sections included: Hero, Features, Testimonials, Pricing, Footer  
- Additional pages such as About, Features, and Contact for multi page navigation  
- Auto generated UI elements using Vercel v0 and refined manually  
- Custom Navbar with working navigation links  
- Designed for clarity, modern aesthetics, and high readability  

## Tech Stack

- Next.js  
- Tailwind CSS  
- Vercel v0 (for UI generation and layout scaffolding)

## Vercel v0 Usage Notes

Vercel v0 was used to generate the initial UI layout for the Hero section, Features section, and Testimonials design. The base structure and component styles were scaffolded via v0 prompts to maintain modern styling and consistent spacing. Typography presets and responsive breakpoints were taken from the autogenerated components. The Pricing section and Footer also draw visual inspiration from v0 generated patterns. All autogenerated code was then manually refactored, cleaned, and componentized to fit the final project structure and improve maintainability.

##Parts generated using Vercel
Hero Section
The hero layout including typography hierarchy, spacing, gradient background, and CTA alignment were generated using Vercel v0 and then refined manually.

Feature Cards
The initial card skeletons, icon placements, and hover interactions were produced using Vercel v0. Final styling and responsiveness were fine tuned in Tailwind.

Testimonial Section
The base structure, avatar placement, and balanced grid layout were generated via Vercel v0 to speed up visual design prototyping.

Color Palette Suggestions
The primary and secondary color directions were suggested by Vercel v0 and integrated into Tailwind for consistency.

Layout Spacing presets
Several spacing and margin patterns were recommended by Vercel v0 during UI generation and then standardized across components.

## Running the Project

Clone the repository and install dependencies:

```bash
npm install
npm run dev
```
The development server will start at:
```
http://localhost:3000
```
##Final Notes

NeuroFlowOS is built with clarity, consistency, and maintainability in mind. All pages follow the same design language and spacing system. The component based structure ensures clean code organization and makes it easy to extend the project with new features or sections in the future.
